### 2️⃣ 创建文章

- 或访问：https://juejin.cn/editor

### 3️⃣ 复制内容

**标题**（选一个）：
```
我用了 1 天做了个电商后台模板，开源了！
```

**正文**：
- 打开 `JUEJIN_ARTICLE.md`
- 复制全部内容
- 粘贴到掘金编辑器

### 4️⃣ 添加截图
- 在对应位置插入截图
- "登录页面" 下方插入登录页截图
- "仪表盘" 下方插入仪表盘截图
- 等等...

### 5️⃣ 添加标签

### 6️⃣ 发布
- 点击 "发布"
- 选择专栏（可选）
- 确认发布
